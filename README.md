<h1 align="center"> Livro: Entendendo Algoritmos 📘 </h1>

By: Aditya Y. Bhargava  

Este repositório reúne anotações, códigos, exercícios e desafios feitos durante a leitura do livro *Entendendo Algoritmos*, uma introdução visual e acessível aos principais algoritmos usados na computação. O objetivo é aprofundar o entendimento por meio da prática, implementando os algoritmos apresentados e explorando suas aplicações no dia a dia da programação.
